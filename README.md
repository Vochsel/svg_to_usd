# SVG to USD

Very WIP...

Convert SVG to a 3D USD scene.

## Features
 * Rectangles
 * Ellipses
 * Polygons
 * Polylines
 * Circles
 * Paths
 * Text
 * Groups
 * Lines
 * Customizable up axis

## Requirements
 * matplotlib
 * fontTools
